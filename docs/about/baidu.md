# 关于我来百度后的一些感受


吃着互联网十几年的红利

会议巨多，不管什么会议，都会拉一帮不相干的人参加

大家开会好像都是一个模式


不像是现在互联网公司该有的活力

规则特别多，开会与自己无关的，必须在线，周报必须写

大家都在按部就班的做自己的事情，没有创新，没有激情，可能是由于老牌互联网公司最终都会国企化，新浪也差不多

感觉做事很强硬，不太会从员工角度思考怎么做更好，大家都比较死板，固执，按部就班地做着没太多意义的事情

百度的口号是【用科技让复杂的事情更简单】，而现在他们做的却是让简单的事情更复杂

权限管理很严格，很多东西都是要经过审批的，审批流程很复杂，审批人很多，审批时间长，审批效率低

来百度这一年，我感觉就是浪费时间，给我职级为T4，我在21年就已经是T4，试用期时不时的就PUA我，说之前存在试用期转不了正的情况来针对我，新人背低绩效还说什么没有主动性，这一年里你们给我分给什么核心的工作了吗，对于员工的职业道路你们是真的一点都不考虑呀，把人骗进来后进开始打击恐吓，世界真TM就是个草台班子，大厂到最后做事风格都会趋于国企化吗？近来一年，我学会的最重要的一点就是要学会对大厂祛魅。

可能是我所在团队的问题，感觉我在的团队缺少对前沿技术发展的敏感度，前端在团队里面的比重也比较弱，没有什么话语权

感觉在百度什么东西都唯职级论，像这种封建糟粕居然还没被淘汰，而且还在如今的互联网大厂中一直盛行着，还一点没有减弱的趋势；职级低的小喽啰能做的事情太少了，职级高的为了维护自己的三瓜两枣有点一言堂的意思，不允许下面的人做自己

感觉百度的大多数员工做事都有点谨小慎微了，没有什么责任感，这可能是大厂的通病吧，作为小螺丝工很难跟管理层共情，也很难跟老板共情


百度这边员工关怀感觉做的特别差，自入职以来从来没有员工沟通这种东西，员工工作中有什么问题也不能及时反馈，部门整体姿态都比较傲慢，总给人一种你爱干不干，不干走人的的傲慢姿态，从不关心员工心理成长。
