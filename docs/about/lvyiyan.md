# 绿漪砚

“素闻沐宸公子练得一手好书法，不知今日在下可否有幸一见？”程璋微作一揖，淡然说道。

沐宸听罢便匀墨掭笔，铺宣行文。须臾之间，笔走龙蛇，墨游缟宣，恍似群马脱缰奔于野，又若孤侠执剑舞于原，笔锋遒劲而健秀，字迹自然而灵动。行文完毕，搁笔停砚，沐宸侧头缓缓说道：“承将军不嫌，俗夫之作，只够让将军耻笑了。”

程璋看罢，惊异之色稍绽腮边便迅转为歆羡之态，侧目投过来的炯炯眼神多了猜不透的深邃。

“世人皆赞王右军之书法有‘翩若惊鸿，矫若游龙’之姿，今日在下所见才称得上是真正的纸上惊鸿，墨间游龙！”

“将军谬赞了。”沐宸星眸半展，语带浅笑。待还欲进言时，忽听书僮阿卜在书房门口说道：“公子，酒已温好。西巷的王伯有急事找你。”

沐宸听罢，转头即看见阿卜强作镇定的神情下惊慌的眼神。沐宸知道，西巷没有所谓的王伯。

沐宸随阿卜来至宅中偏隅，阿卜密声向沐宸说道：“刚才趁公子与程璋交谈之际，我发现我们宅子四周暗藏有数十名杀手。”

沐宸眉心微凝，忖思片刻，随即朗声说道：“我知道了，让王伯明日再来吧，且让我今日与程璋将军一醉方休！”

影滤翠竹，色侵黛屏。胧月出岫，雅阁泛香。程璋手执玉樽，朗声说道：“邻翁即访，何不约来同饮？”

“哦，王伯邀我做他家公子教书先生，我约他明日再作商榷，他就先走了。”沐宸挽袖持樽，神色恬淡，继续笑道：“程将军今日不期造访，在下略备旧醅，望将军不弃。”

二人对饮刚罢，程璋即赞道：“浓而不烈，香郁而味醇，真乃好酒！沐宸公子看来不止是再世书圣，还是再世杜康啊！”一语言罢，阵阵爽笑绕帘抚栊，遁入清寂轩院。程璋继而说道：“听公子口音像是南方人，不知祖籍在哪儿？”

“先祖世代安居姑苏，我因尚自在悠闲，两年前与书僮游学至京，故暂居于此。”沐宸神怡态和，缓缓说道。

“鄙人曾闻姑苏有一沐氏望族，为制砚世家。它家所制墨砚细润如玉，滑腻如瓷，发墨而利笔，堪为砚中极品，不知公子知不知？”

樽酒突然一阵轻晃，沐宸的头影被摇成一圈圈碎粼。沐宸深吸一口气，再次拿起玉樽，缓缓而饮，自然笑道：“怎能不知！姑苏沐氏乃雍华富贵之族，我等只能垂涎称羡了。”

“难道公子祖上不是这一族吗？”

“我乃村野愚夫，岂敢玷他家门楣！只是碰巧同姓而已。”沐宸饮罢，眼角叠笑。

程璋趁饮酒之际看了沐宸一眼，眉间闪过瞬时的迟疑与忧虑。略神思后寂然说道：“只是五年前它家突然惨遭屠门，不禁令人扼腕！”

“世事无常，岁月多谲。富贵难永继，繁华岂恒昌！”

“公子高见，在下受教了。”程璋看着对面沐宸淡定的神情，眉心微蹙，执樽默默啜饮。

疏影横斜，月入黄昏。阿卜燃烛点灯，取麝焚香，不一会儿，室内一派通明。

程璋环视四周，目光游移之际却突然在书房门口停住了。“我刚才见公子书房案前的墨砚通体碧绿沉静，砚身纹理绮丽优美，想来应是砚中极品，可否再借我一观？”称璋突然说道。

“将军愿观，取来便是。”沐宸侧目望向阿卜，说道：“阿卜，快将我书房中绿漪砚取来将军瞧瞧！”

阿卜神色突然紧绷，眸中闪过寒光。但和沐宸对视一眼后便徐步行至书房，将绿漪砚奉至程璋席前。

“绿纹如漪，果如其名啊！听闻那制砚的姑苏沐家也有一块天下无双的沉绿宝砚，可是这块？”程璋问道。

“将军说笑了，我这绿砚是在杭州云游时从一穷儒手中购得，实为凡品，只是乍眼之下，能够乱真罢了。”

程璋托砚于手，佯作观赏。突然左手急缩，砚落于地，只听见啪嚓一声，漪断碧残。

阿卜突然杏目圆睁，健硕的手臂上布满红色的血络。

沐宸眸光一凛，深吸一气，箭目看向程璋。

程璋看了沐宸一眼，佯作惶恐呼道：“在下不小心摔碎了绿漪砚，望公子见谅！”

“如公子所见，破砚碎石一堆，哪是什么珍宝！”沐宸镇静说道，“阿卜快将它收拾干静扔掉，免得扰了程将军饮酒的雅兴！”

程璋看了看席畔安静躬身收拾破砚的阿卜，再转目看了看对面神色淡然的沐宸，饮下半杯残酒，竟也不解其味。最后，复饮了几杯，仍不解其味，便辞别沐宸，索然离去。


“公子何不杀了那厮？”阿卜悄声问正在书桌旁拼接碎砚的沐宸。

“他只是王丞相的走狗，杀了他只会引火自焚。”沐宸望着窗外孤月，肃然说道：“看来那奸相已经发现我们了，今晚我们就得离开京城。”

阿卜看着书桌上残缺的绿漪砚，寂然说道：“五年前我与公子幸得逃脱那奸相屠杀，这几年我们带着此砚东奔西走，我们沐氏一族的杀戮也是因此砚而起，只是阿卜不知，此砚除石质精良外，有甚特别？”

沐宸将残砚倒置，砚底镌有“庚寅年制”四字。他滴一滴清水于字上，深红字体变得稍大了些，上面出现了一些清晰的细小纹络。

“滴水有窥细放大之效，这四字放大后就是一幅藏宝图。当年先祖请巧匠将此藏宝图以＇庚寅年制＇四字样镌刻于砚底，故我族世代视此砚为珍宝。那贼相知此绿漪砚中藏有我族累世所积金银，便派人来夺，故此引发一番杀戮。”沐宸长叹一气继而说道：“那程璋愚士，竟以为藏宝图藏于砚中，故运功摔碎此砚。幸在你我未让怒火乱了阵
脚，不然我们今晚就死无藏身之地了。”

孤月遁影，天地无光。沐宸和阿卜收拾好包袱，便望向苍茫的凄寂黑夜匆匆离去。
