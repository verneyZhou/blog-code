# 全栈项目开发实录：vue-node-admin

这里是我从学习到整理，爆肝两个多月完成的全栈项目学习记录，主要包括阿里云服务器购买、服务器环境搭建，再到后端环境配置等等，也是第一个从0到1独立完成的全栈项目。

- 前端：vue + element-ui
- 后端：node + express
- 服务器：nginx
- 数据库：mySql
