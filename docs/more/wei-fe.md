# 微前端服务


## 简介

微服务架构：微服务架构可以将一个应用分成若干个更小的服务，这让整个开发过程具有很高的敏捷性和可拓展性。

它更多的是单页应用并有多框架隔离的需求，然后做出微前端这样一个技术方案。
