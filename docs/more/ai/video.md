# AI原创短片创作实操笔记

> 最近Seedance2.0出来了，尝试了一下，效果还不错，可以说是AI视频模型中第一梯队的产品了。然后就萌生了用它做一个AI微电影的想法，下面这个链接就是最终做出来的成品，目前发布到B站；虽然画面上有些地方还是会有AI感，但感觉总体效果还不错，这里记录下我的创作过程。

## 文生剧本
> AI生成视频比较吃提示词，提示词的好坏直接决定最终生成视频的质量，就算是Seedance2.0这样先进的模型，如果提示词写得不够准确，最终的效果也差强人意，所以首先需要生成专业的分镜剧本。

上面的AI短片是基于我大学时写的一篇微小说[《绿漪砚》](/about/lvyiyan.html)改编的，如果没有现成的文本故事，可以让AI根据你输入的主题，输出一个简单的故事结构，再根据这个故事结构生成完整的故事。

我首先是直接把我的小说转换为`txt`文件，然后扔给DeepSeek，让它生成分镜剧本：

```

```

## 人物角色生成
> 在AI视频中比较大的一个问题就是角色的一致性了，Seedance2.0这次出了`全能参考`模式，可以上传人物进行参考，所以首先需要生成人物的形象图

- 像在我的小说中主要有【沐宸】、【程璋】、【阿卜】三个角色，所以我需要通过DeepSeek或Gemini生成这三个角色形象的提示词：
```
现在我需要你结合我的小说内容及你给出的分镜描述，帮我生成小说中三个人物：沐宸、程璋、阿卜三个人物的人物形象提示词
```
> 生成完成后看下描述是否符合预期，是否需要调整；不符合预期可以让AI再按你的想法微调下，直到符合预期为止。

- 之后就开始复制人物提示词，打开[即梦](https://jimeng.jianying.com/ai-tool/generate)，选择图片生成，输入人物提示词，生成人物形象图；我这里以【沐宸】为例：
> 也可以用NanaBanana生成人物三视图，效果也不错

## 文生视频

- 即梦选择视频生成，选择Seedance2.0的`全能参考`，直接复制生成的分镜剧本提示词：
```
镜头1：
  景别：特写
  画面描述：一只手轻抚宣纸，指尖修长，骨节分明。宣纸上墨迹未干，“惊鸿”二字遒劲灵动
  对白/音效：窗外竹叶沙沙声（环境音）
  画面提示词：古风摄影级别，特写手部，修长手指轻抚宣纸，墨迹未干的书法“惊鸿”二字，遒劲灵动，自然光影，青绿色调，电影质感，景深效果，4k高清

镜头2：
  景别：中景
  画面描述：沐宸立于案前，青衫素袍，眉目清朗。他抬眸看向对面	
  对白/音效：程璋（微作一揖，淡然）：素闻沐宸公子练得一手好书法，不知今日在下可否有幸一见？
  画面提示词：古风摄影级别，中景，古代书房，青衫公子立于书案前，眉目清朗，抬眸看向镜头外，窗外竹影摇曳，自然光线，电影质感，柔和光影，4k高清

镜头3：
  景别：跟拍
  画面描述：沐宸匀墨——墨条在绿漪砚中缓缓旋转，墨汁浓稠如漆。他掭笔、铺宣，动作行云流水
  对白/音效：沐宸（浅笑）：承蒙将军厚爱。
  画面提示词：古风摄影级别，跟拍镜头，青衫公子研墨，墨条在青绿色砚台中旋转，墨汁浓稠，掭笔动作优雅，书案上的宣纸，光影流转，电影质感，4k高清

上面镜头中的【沐宸】人物形象参考@1，对白音色参考@2；【程璋】人物形象参考@3，对白音色参考@4
```
> 输入提示词后，选择参数，检查后有没有什么问题，就可以点击提交，开始生成了

因为目前Seedance2.0生成的视频最长支持15s，需要自己权衡是输入分镜剧本的一个镜头还是多个镜头，需要根据剧情需要进行调整，这里就看个人喜好了；

上面我传了音频，感觉效果一般，所以我打算后期剪辑的时候再配音，后面的镜头我就没有传入音频参考，只传入了图片参考。


当然分镜剧本只是参考，生成过程中也是抽卡，也会遇到自己不满意的，这个需要自己调整提示词，或者变换下剪辑思路，这些都是看个人。等到所有你想要的分镜都生成完成后，就可以下载下来进行编辑了

视频导出的时候记得**提高帧率和提升分辨率**！！！我选择的是`30fps`和`HD`超清


## 后期剪辑
> 导入视频到剪辑软件，我这里用的是剪映（字节全家桶了属于是...）

这里就需要有点视频剪辑基础了，导入分镜视频片段，按照自己的创作思路进行剪辑


### 人物配音

接下来是最重要的人物配音问题，生成的AI视频里面会默认有旁白，你可以选择自己满意的视频人物声音，先单独导入剪映，导出音频文件就得到该人物的声音；之后选择剪映的`克隆音色`，导入声音文件，就能得到该人物的克隆声音了。

<img :src="$withBase('/images/ai/video06.png')" width="auto"/>

之后添加字幕，选择该人物旁白`朗读`，就完成配音，然后调整音频轨道跟人物嘴型匹配就行。

之后的工作就是微调...，调到自己满意为止...也是比较耗时间的抠细节的工作...

添加字幕，添加配音，剪辑，添加bgm，环境音效，转场，字体效果...这些东西基本都是熟能生巧...

至此，一个AI原创故事短片就做成了...虽然有AI赋能，但感觉自己做还是会花很多时间😄...





## 其他

- AI在生成视频时，第一帧往往是高清参考图，后面才是生成的内容，默认连在一块，清晰度和色彩会有跳变，因此需要把每段视频的第一帧给切掉。

- 画质跃迁：在导出前，千万别忘了在极梦后台点击**“补帧”和“提升分辨率”**。将 30 帧提升至 60 帧，视频的丝滑程度会发生质变。


### AI生成广告短片

1. 生成分镜提示词：首先在[Gemini](https://gemini.google.com/app)中输入以上提示词，得到分镜图提示词。
``` 
你是一个专业的广告片导演，有25年行业经验，请给我写一个【AI眼镜】的商业广告分解提示词，分9个分镜，比例3:2，4k。 
```

2. 得到提示词后，看下是否满足需要，可以调整细节；如果没有调整直接输入：`帮我按照上面生成的分镜提示词生成九宫格分镜图吧`；然后选择它的`Nana banana`制作图片，直接生成九宫格分镜图。

> [Lovart](https://www.lovart.ai/)里面也可以使用`Nana banana`来生成分镜图，也可以使用Kling来生成视频。

3. 生成九宫格分镜图后，下载图片；之后进入[即梦](https://jimeng.jianying.com/ai-tool/generate)，选择`视频生成`，选择`Seedance2.0`，上传上一步生成的九宫格分镜图，输入提示词：`这个是我生成的以「AI眼镜」为主题的广告短片的九宫格分镜，你帮我按照这个分镜图生成一条广告短片吧`；之后点击`生成视频`，即可生成广告短片。



## 提示词

### 文生视频

镜头1 | 镜头+角色+动作+台词
> 例：镜头1|近景•林缚刑场抬头，大雪砸在他血污的脸上，冲刷出棱角分明的轮廓，猩红的眼死死盯着远处马车，沙哑嘶吼：“苏摇扣、林婉柔，若有来世，我定要你们血债血偿！”

最核心结构（万能公式）：主体+动作+场景+光影+镜头语言+风格+画质+约束
> 例：一位年轻女生在海边慢走，微风拂动头发，微笑看向镜头，黄昏暖光，4K高清，电影感，稳定运镜，画面流畅不抖动，细节清晰。


【主体】+【动作/情节】+【镜头运动】+【画面风格】+【光影氛围】+【参数控制】




### 镜头语言

- 近景 / 中景/伞景 / 特写                                                                                                               
- 缓慢推镜 / 轻微拉远 / 平稳構移/ 环绕半圈 / 轻微环绕 / 平稳跟拍
- 固定镜头、手持稳定、无抖动、丝滑流畅 
> 例：中景，缓慢推镜，稳定跟拍，画面丝滑不卡顿 

角色/人物稳定 （防变脸、防变形），加约束词：
- 五官清晰、面部稳定、不扭曲、不恋形
- 人 体结构正常、比例自然、动作不僵硬                                                                              
- 同一角色、服装一致、发型不变

画面质量 （必加、 提升清晰度）：
- 4K、超高清、细节丰富、锐度清晰                                                                                      
- 电影质感、色彩自然、光影柔和                                                                                           
- 无模糊、无重影、无闪烁、画面稳定 


风格＆氛围 （快速出片）
- 治愈清新、唯美文艺、日系清新、韩系氛围感                                                                              
- 赛博朋克、暗调高级、复古胶片、梦幻柔光                                                                      
- 极简干净、高级质感、直实摂景


- 唯美人像短视频：一位年轻女生在林间缓慢行走，微风轻拂头发，自然微笑，暖光光影，中景，缓慢推镜，画面流畅稳定，4K高清，电影感，面部清晰不变形，人体结构正常，细节丰富
- 氛围感风景：海边日落，海浪轻拍沙滩，镜头缓慢横移，暖橙色调，治愈清新，画面丝滑，4K超高清，无闪烁无重影
- 图生视频，首帧固定：基于参考图保持人物样貌与服装一致，动作缓慢抬手转身，自然流畅，不僵硬不变形，稳定运镜，高清细节，电影质感 


### 分镜

- 直接提示词描述生成视频
例：西红柿炒鸡蛋教程
分镜1 | 特写.人物的双手入镜
分镜2 | 中景.在锅里倒油后倒入鸡蛋液
分镜3 | 中景.等待鸡蛋熟了之后
分镜4 | 特写.人物出锅

- 可以先生成分镜图，再根据分镜图生成视频


### 视频编辑

把这个视频@[视频1]的背景替换成这张图片@[图片1]，仅替换背景画面保持原有的运镜


### 镜头与动作迁移

参考图@[图片1]的动漫角色形象，让他住在图@[图片2]的环境中，完全参考视频@[视频1]的运镜效果；


镜头语言：特写、中景、远景、第一人称视角。
环境描述：雷雨交加、古代集市、沙漠、海面。
动作细节：翻腾旋转、极速飞驰、血水冻凝。
风格指定：宫崎骏画风、电影质感、黑白漫画质感。
引用功能（高级用法）：使用ref或@来指定参考图和参考视频，这是保持角色一致性和动作迁移的关键。
